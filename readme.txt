== Description ==
m:lightbox is a modern, minimal lightbox. Insert links to images/image attachments into posts and pages and the image will be displayed in a lightbox automatically.**

Features:
* Fully-responsive image lightbox
* No configurable options
* No admin links
* No pro version

This lightbox plugin is wraps imagelightbox (www.osvaldas.info) by Osvaldas Valutis into a plugin
